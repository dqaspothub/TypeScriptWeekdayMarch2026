## 1️⃣ What is a String in JavaScript?

A **string** is a sequence of characters used to store and manipulate text.

Examples:

```javascript
"Hello"
'JavaScript'
`Welcome to JS`
```

Strings can contain:

* Letters
* Numbers
* Symbols
* Spaces

---

## 2️⃣ Ways to Create a String

### 1. Using Double Quotes

```javascript
let name = "John";
```

### 2. Using Single Quotes

```javascript
let city = 'Bangalore';
```

### 3. Using Backticks (Template Literals) ✅ (Most powerful)

```javascript
let course = `JavaScript`;
```

---

## 3️⃣ Template Literals (Backticks ` `)

Used for:
✔ String interpolation
✔ Multi-line strings

### Example:

```javascript
let name = "Alex";
let age = 25;

console.log(`My name is ${name} and my age is ${age}`);
```

📌 **Output:**

```
My name is Alex and my age is 25
```

---

## 4️⃣ String Length

Use `.length` property:


let str = "JavaScript";
console.log(str.length);


📌 Output:

```
10
```

---

## 5️⃣ Accessing Characters

Strings are **indexed** (index starts from 0)

```javascript
let str = "Hello";

console.log(str[0]); // H
console.log(str[4]); // o
```

---

## 6️⃣ String Immutability (Very Important ⭐)

👉 Strings **cannot be changed** once created.

```javascript
let str = "Hello";
str[0] = "Y";

console.log(str); // Hello (unchanged)
```

To change a string, you must create a **new one**.

---

## 7️⃣ Common String Methods (Very Important for Interviews)

### 🔹 `toUpperCase()` / `toLowerCase()`

```javascript
let str = "JavaScript";

console.log(str.toUpperCase()); // JAVASCRIPT
console.log(str.toLowerCase()); // javascript
```

---

### 🔹 `trim()`

Removes spaces from start & end:

```javascript
let str = "  Hello World  ";

console.log(str.trim());
```

---

### 🔹 `includes()`

Checks if text exists:

```javascript
let msg = "Welcome to JavaScript";

console.log(msg.includes("Java")); // true
```

---

### 🔹 `startsWith()` / `endsWith()`

```javascript
let file = "index.html";

console.log(file.startsWith("index")); // true
console.log(file.endsWith("html"));    // true
```

---

### 🔹 `indexOf()`

Returns position of character or word:

```javascript
let str = "JavaScript";

console.log(str.indexOf("S")); // 4
```

Returns `-1` if not found.

---

### 🔹 `slice()`

Extracts part of string:

```javascript
let str = "JavaScript";

console.log(str.slice(0, 4)); // Java
console.log(str.slice(4));    // Script
```

---

### 🔹 `substring()`

Similar to slice (no negative index):

```javascript
console.log(str.substring(0, 4)); // Java
```

---

### 🔹 `replace()`

Replaces text:

```javascript
let msg = "I love Java";

console.log(msg.replace("Java", "JavaScript"));
```

---

### 🔹 `split()`

Converts string to array:

```javascript
let str = "apple,banana,orange";

console.log(str.split(","));
```

📌 Output:

```javascript
["apple", "banana", "orange"]
```

---

## 8️⃣ String Concatenation

### Using `+`

```javascript
let first = "Hello";
let second = "World";

console.log(first + " " + second);
```

### Using Template Literals (Recommended ✅)

```javascript
console.log(`${first} ${second}`);
```
