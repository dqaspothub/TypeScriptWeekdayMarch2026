JavaScript String Methods

Visual & Interactive Learning Experience

📏
length
Returns the number of characters in the string
str.length
Enter a string to see the visual representation
🎯
charAt()
Returns the character at the specified index
str.charAt(index)
Enter a string and index to highlight the character
✂️
slice()
Extracts a section of a string and returns it as a new string
str.slice(start, end)
Enter values to see the sliced portion highlighted
⬆️
toUpperCase()
Converts all characters to uppercase
str.toUpperCase()
Enter text to see the transformation
⬇️
toLowerCase()
Converts all characters to lowercase
str.toLowerCase()
Enter text to see the transformation
🔪
split()
Splits the string into an array of substrings
str.split(separator)
Enter a string and separator to see the array
➕
concat()
Joins two or more strings together
str.concat(str2, str3, ...)
Enter strings to see them combined
🔍
indexOf()
Returns the index of first occurrence
str.indexOf(searchValue)
Search for a character or substring
🔄
replace()
Replaces the first occurrence of a substring
str.replace(search, replacement)
Enter values to see the replacement
✨
trim()
Removes whitespace from both ends
str.trim()
Enter a string with leading/trailing spaces
✅
includes()
Checks if string contains a substring
str.includes(searchString)
Search for a substring
🔁
repeat()
Repeats the string a specified number of times
str.repeat(count)
Enter a string and count
